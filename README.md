<h3>Install </h3>
<ul>
    <li>１．Docker立ち上げ</li>
    <dl>docker-compose up -d</dl>
    <li>２．appに入る</li>
    <dl>docker-compose exec app bash</dl>
    <li>３．各種インストール</li>
    <dl>
        npm install<br>
        composer install
    </dl>
    <li>４．npm実行</li>
    <dl>
        php artisan key:generate<br>
        npm run watch(npm run dev)
    </dl>
    <li>５．DB接続</li>
    <dl>
        cp .env.example .env<br>
        php artisan migrate:fresh --seed
    </dl>
</ul>
